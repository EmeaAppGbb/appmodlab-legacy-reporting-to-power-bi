# Legacy Reporting to Power BI

This lab demonstrates modernizing Crystal Reports and SSRS to Power BI for Clearwater Health System.

## Structure
- `CrystalReports/` - Sample Crystal Reports (.rpt files)
- `SSRS-Reports/` - Sample SSRS reports (.rdl files)
- `SQL/` - Database objects (stored procedures, views, functions)
- `DataSources/` - Connection configurations
- `Subscriptions/` - Email distribution settings

See [APPMODLAB.md](APPMODLAB.md) for complete lab documentation.
